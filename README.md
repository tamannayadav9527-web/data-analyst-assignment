#  Data Analyst Assignment

## Overview
This project involves analyzing a transactional dataset using Google Sheets. The objective was to apply data cleaning, transformation, and analytical techniques to derive meaningful insights.

---

## Tools Used
- Google Sheets
- Excel (.xlsx)
- Pivot Tables
- Spreadsheet Functions

## Key Tasks Performed
- Calculated Net Amount, Tax Amount, and Final Amount using conditional formulas
- Handled inconsistent data formats (dates, discount %, text case)
- Created Pivot Table for Region-wise analysis
- Built Monthly Trend with:
  - Monthly Revenue
  - Cumulative Revenue
  - MoM Growth %
- Identified highest growth month and customer behavior patterns

---

## Key Insights
- Highest MoM growth observed in **Jan-2025 (~807%)**
- No customers found in both Active and Cancelled/Churned categories
- Data inconsistencies were identified and handled to ensure accurate analysis

---

## Notes
All calculations are dynamic and formula-driven. No values were hardcoded.
